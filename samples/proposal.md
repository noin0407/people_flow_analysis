# 提案書：データ駆動型アーケード・モビリティ「SENDAI FLOW CART」の導入について

1.  導入の背景と目的

仙台市中心部の商店街（アーケード街）は、全長約 1.5km に及ぶ貴重な歩行者空間であるが、以下の課題を抱えている。

- 回遊性の低下： 仙台駅から一番町・定禅寺通への移動距離が長く、高齢者や観光客の回遊が阻害されている。
- 滞留の偏り： 特定の地点（駅前等）に人が集中し、アーケード奥部への人流波及が限定的である。
- 歩行環境の質的向上： 単なる「通路」から、快適な移動体験を提供する「滞留空間」への転換が求められている。

  2.事業概要：SENDAI FLOW CART(仙台・フロー・カート)

- 人流解析アプリ「SENDAI FLOW」のリアルタイムデータを活用し、アーケード内を運行する低速電動カート（グリーンスローモビリティ）の最適運用を行う。
- 運行形態： 決まったルート内での乗り合い運行（時速 5km 程度）。
- データ連携： 1 時間ごとの人流予測に基づき、車両の配備・速度・運行間隔を動的に制御する。

3. 本提案の独自性（データ活用による効率化）

   定時・定路線運行とは異なり、独自開発の分析基盤を活用した「オンデマンド・マネジメント」を行う。

- 車両配置 : 固定配置（無駄な空車走行）/ 需要予測に基づく事前配置（プレ・ポジショニング）
- 運行速度 : 一定（混雑時に危険）/ 人流密度（Occupancy）に連動した速度制限
- 停留所 : 物理的な看板を設置 / 滞留データに基づく「バーチャル停留所」の最適化
- 評価指標 : 乗車人数のみ / 移動による周辺店舗の売上向上（回遊性向上）の可視化

4. 期待される効果経済の活性化： カートによる移動負荷の軽減により、一番町エリアや裏路地（横丁）への訪問客数を約 15%向上させる（推計）。

- バリアフリーの実現： 高齢者や障がい者の移動自由を確保し、健康増進・社会参画を促す。
- スマートシティのモデル化： 仙台独自のオープンデータと学生の IT 技術を融合させた、次世代都市交通の先駆けとする。

5. 今後の展開（ロードマップ）

- フェーズ 1： 「SENDAI FLOW」アプリによる現況人流の精密分析とシミュレーション。
- フェーズ 2： 特定区間における期間限定の社会実験（実証運行）。
- フェーズ 3： 商店街振興組合および民間企業と連携した本格運用・収益化モデルの構築。ブ
